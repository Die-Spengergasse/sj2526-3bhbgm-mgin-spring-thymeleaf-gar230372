# Instructions to use the application 
* start the SprinngThymeleafApplication (so that it´s running)
* use the browser to open http://localhost:8080
* on the page, click on the link called "Einsicht zur Obstliste"
* you will get forwarded to a new page, where you can see all fruits that had been added to the database before in a table

Functions:
* adding a new fruit: click on the link "new obst" below the table, you´ll get forwarded to a page where you have to put all the data of the fruit in the fields, after that click on "save" (then you get back to the table)
* editing a fruit: in the table you can click on "edit obst" to change its values, that´s also happening on a different site, after changing the data you´ll get back to the table
* deleting a fruit: in the table you can click on "delete obst" to delete it from the database
